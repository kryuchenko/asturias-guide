# Asturias Guide

Справочный сайт для русскоязычных жителей Астурии, Испания.

Astro + MDX, деплой на GitHub Pages.

## Разработка

```sh
bun install
bun dev        # localhost:4321
bun build      # ./dist/
```

## Деплой

Автоматический через GitHub Actions при пуше в `main`.

Сайт: https://kryuchenko.github.io/asturias-guide/
